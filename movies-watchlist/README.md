# Movie Watchlist

A small client-side Movie Watchlist application that demonstrates functions and arrays in JavaScript.

Features
- Add movies with optional category and rating
- Display the watchlist dynamically
- Remove movies when watched
- Edit movie title (simple prompt)
- Search/filter movies by title or category
- Persistence using localStorage

Files
- `index.html` - Main HTML file
- `styles.css` - Simple styling
- `script.js` - JavaScript logic (add/display/remove/search/persistence)

How to run
1. Open `index.html` in your browser (double-click or use Live Server extension).
2. Add movies using the input fields and click "Add Movie".
3. Use the search box to filter the list.
4. Click "Watched" to remove an item.
